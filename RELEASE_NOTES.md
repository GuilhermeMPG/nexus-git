## Novidades

- **Vínculos:** agora também dá pra selecionar **1 branch e vários cards** de uma vez (antes só
  era possível 1 card + várias branches). Tem um seletor de modo acima das colunas.
- **Corrigido:** publicar Vínculos/Erros de um segundo projeto podia sobrescrever a página de
  Wiki de outro projeto. Cada projeto agora tem um título de página próprio — veja
  Configurações se você usa mais de um projeto.
- **Novo:** o app confere periodicamente se há uma versão nova no GitHub e avisa na barra
  lateral, com link direto pra baixar.
- Clique na versão (canto superior da barra lateral) para ver este resumo de novidades a
  qualquer momento.
